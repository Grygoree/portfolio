Portfolio

Rapid Task Switcher

Rapid Task Switcher is a browser based game where a player must juggle a number of microgames in order to win. It is my reimagining of an old Flash game called Multitasker. This project best represents my code habits and practices that I'm proud of. Since the game was browser based, I used a lot of vanilla tech instead with a helping of DOM manipulation instead of an existing game framework like Phaser.io or Three.js

Technology Used

Vanilla HTML5, CSS, and Javascript

Deadringer

Deadringer is a proof of concept dead man's switch messaging system. I used SQL and Flask API design in order to create a messaging system so that users can have more control over message scheduling. It also made implementing a receipt system easier. The UI was implemented using React in order to compartamentalize individual messages.

Technology Used

Python, Flask, Javascript, React

Github Frontend Github Backend Demo

Melopolo

Melopolo is a global audio quiz game where users can record an audio snippet using a microphone, and give the world a quick 10 second quiz. I utilized MongoDB and Node to create a backend API that handled raw audio data. From what I learned on the frontend, getting user media input was an interesting but satisfying challenge.

Technology Used

Javascript MERN, MongoDB, Express, React, Node
